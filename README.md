# Rajmata Enterprise — Business Website

A professional bilingual (Hindi + English) business website for **Rajmata Enterprise**, a wholesale supplier of disposable items, cleaning products, and packaging materials based in KPHB Colony, Hyderabad.

## Key Features

- **Bilingual UI** — Full Hindi and English support with a live language switcher (persisted via localStorage)
- **5 Pages** — Home, Products, About Us, Contact Us, Online Order Booking
- **WhatsApp Integration** — Floating WhatsApp button and deep-links on all forms and product cards
- **Product Gallery** — 10 products with category filter (Bags, Disposables, Cleaning, Packaging)
- **Contact Form** — Sends inquiry as a formatted WhatsApp message
- **Order Booking Form** — Multi-product checkbox form, submits as a structured WhatsApp message
- **Google Map** — Embedded map showing KPHB Colony, Hyderabad
- **Mobile Responsive** — Hamburger menu, responsive grids, touch-friendly UI

## Color Theme

- **Royal Blue** (`#1e3a8a`) — Primary brand color
- **Orange** (`#f97316`) — Accent and CTA color
- **White / Off-White** — Backgrounds

## Tech Stack

- [TanStack Start](https://tanstack.com/start) — Full-stack React meta-framework
- [TanStack Router](https://tanstack.com/router) — File-based type-safe routing
- [Tailwind CSS v4](https://tailwindcss.com/) — Utility CSS (via Vite plugin)
- [Lucide React](https://lucide.dev/) — Icons
- [Vite](https://vitejs.dev/) — Build tool
- Hosted on **Netlify**

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
